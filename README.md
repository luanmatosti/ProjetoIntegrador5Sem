## Getting Started

Assim que clonar o repositório em sua máquina realizar esses passos:

1 - Abrir o pasta do projeto onde você criou.
2 - Clicar com o botão direito e clicar em "GIT Bash Here"
3 - Escrever "flutter pub get"
4 - Pronto, seja feliz :)
